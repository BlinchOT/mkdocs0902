# Лабораторная работа №1: Создание статического сайта на MkDocs

## Цель работы
Освоить создание статического сайта с помощью MkDocs и публикацию на GitHub Pages.

## Задание
1. Создать репозиторий
2. Настроить GitHub Pages из папки `/docs`
3. Развернуть сайт

## Код / Команды
```bash
mkdocs new source
mkdocs build -d ../docs
git add .
git commit -m "Deploy site"
git push
```
